# 使用GDB调试Nginx

*. 运行gdb
```shell
$ sudo gdb -q -tui
```

*. 启动nginx（若已经启动则可以忽略）
```shell
#和平时启动nginx一样，在前面增加shell表示用shell运行
（gdb） shell nginx
```

*. attach Nginx worker进程
```shell
#找到 Nginx worker进程的进程号
（gdb） shell pidof nginx
（gdb） attach pid
```

*. 打断点
```shell
（gdb） b function_name
```

*. 开始执行
```shell
（gdb） c
```

*. 单步调试
```shell
（gdb） n
```

*. 单步跟踪进步调试
```shell
（gdb） s
```

*. 删除断点
```shell
（gdb） d
```
